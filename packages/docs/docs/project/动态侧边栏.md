# 动态侧边栏

电商项目的侧边栏针对不同的页面，有不同的功能需要呈现，统一在  `components/quick-link/index.uve` 组件下管理。

## 场景需求
- 如果非特殊页面，则侧边栏显示默认的功能如：['Chat', 'FansGroup', 'QrCode', 'BackTop']
- 如果是特殊页面，则需要显示特有的功能如装修工具页：['Chat', 'QrCode', 'ToolVideo', 'BackTop']
- 特别页面根本无需显示侧边栏： [] 传入空数组

## 实现思路
因为业务场景是根据页面去区分的，所以在路由配置是最佳的，我们约定在路由的`meta`字段里添加`quickLink`字段如下:
```javascript
{
    name: 'tools',
    path: '/tools/:queryStr?',
    component: Tool,
    meta: {
        lazyMeta: true,
        showNav: true,
        link: '/tools',
        title: '装修工具',
        order: 7,
        quickLink: ['Chat', 'FansGroup', 'QrCode', 'ToolVideo', 'BackTop'],
    },
},
```

然后在组件里监听当前路由，得到对应的配置

```javascript
watch: {
    $route: {
        handler(v) {
            if (this.fromTaobao) {
                this.child = ['TaobaoChat', 'BackTop'];
            } else if (v.meta && v.meta.quickLink) {
                this.child = v.meta.quickLink;
            } else {
                this.child = defaultLinks;
            }
        },
        immediate: true,
    },
},
```

然后再通过`this.child`得到我们真正需要的组件并通过`component`的`is`属性渲染出来。

```vue
// 得到真正的组件 childrens
computed: {
    childrens () {
        return this.child.reduce((p, n) => {
            if (links[n]) {
                p.push(links[n]);
            }
            return p;
        }, []);
    },
},

// 渲染组件
<quick-link v-if="childrens.length > 0">
    <component
        :is="component"
        :key="i"
        v-for="(component,i) in childrens">
    </component>
</quick-link>
```
